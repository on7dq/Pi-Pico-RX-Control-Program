# Pi-Pico-RX-Control-Program
A Windows program to control a Pi Pico RX

<img width="654" height="516" alt="image" src="https://github.com/user-attachments/assets/4764dea5-9692-4033-951e-2b106a611738" />


A Pi Pico RX Control Program with direct frequency entry, mode selection, and up to 135 presets.
The program can also be used to create a headless receiver (no display or buttons needed), see included manual.

Author of the program is **Gil, ONL12523**, who also programmed the **OST Morse Box** and the **49'er PLL** projects.
Documentation was done by **Luc, ON7DQ**, yours truly ;-)

Enjoy the program, we hope you find it useful !
